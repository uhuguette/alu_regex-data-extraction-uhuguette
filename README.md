 Regex Data Extraction and Secure Validation (JavaScript)

This project is a JavaScript program that extracts structured info from raw text using regular expressions. The input simulates data from an external API and may include realistic variations and unsafe content.

 What the program does

- Reads raw text from input.txt  
- Extracts:Emails, URLs, Phone numbers,Credit card numbers (masked), Time values (12-hour and 24-hour formats)  
- Filters out unsafe/malicious content  
- Prints extracted data to the terminal  
- Saves results in JSON (output.json) and text (output.txt)
